## [1.2.0] — 2026-03-13

### Добавлено
- **Напоминания за 1 час до занятия** — автоматическая отправка через WA/TG/Email всем родителям группы (`src/reminders.js`)
- **CSRF-защита** — проверка Origin-заголовка для POST/PUT/DELETE на `/api/*`
- **Timing-safe аутентификация** — `crypto.timingSafeEqual` вместо `===` для сравнения паролей
- **HTTPS redirect** — автоматический редирект HTTP→HTTPS при `BASE_URL=https://...`
- **Graceful shutdown** — корректная обработка SIGTERM/SIGINT, остановка таймеров
- **Docker hardening** — non-root пользователь (`node`), `HEALTHCHECK`, `.dockerignore`

### Изменено
- **Ребрендинг на частные кружки**: `урок` → `занятие`, `учитель` → `педагог`, `класс` → `группа`
- `server.js` — интеграция `reminders.start()` при старте, `reminders.stop()` при shutdown
- `email.js` — обновлены тексты уведомлений
- `telegram.js` — обновлены команды бота (`/start`, `/stop`, `/help`)
- `validate.js` — `Время начала занятия` вместо `урока`
- `public/index.html` — обновлены плейсхолдеры, иконки, метки аудита
- `public/help.html` — обновлены инструкции для родителей
- `.env.example` — обновлён пример домена
- `Dockerfile` — добавлены `USER node`, `HEALTHCHECK`

## [1.1.0] — 2026-03-11

### Добавлено
- **SQLite вместо JSON** — база данных переведена на `sql.js` (чистый JS, без нативной компиляции)
- **Группы/классы** — управление несколькими группами, фильтрация во всех вкладках
- **QR группы + PIN** — один QR на кабинет, индивидуальный PIN для каждого ученика
- **4 WhatsApp провайдера** — Baileys (бесплатный), Green API, Twilio, WWebJS
- **Email уведомления** — SMTP через nodemailer (Gmail, Yandex, Mail.ru)
- **Кабинет родителя** — `/parent/:token` с историей посещений и статистикой
- **Уведомление об опоздании** — автоматический расчёт по расписанию группы
- **Экспорт в Excel** — `.xlsx` с фильтром по группе и дате, форматированием
- **Статистика ученика** — сводка за 6 месяцев (всего/вовремя/опозданий/%)
- **Telegram Webhook** — автоматическая регистрация при HTTPS-домене (fallback: Long Polling)
- **Дашборд** — вкладка «Сегодня» с живой сводкой + онбординг-чеклист + график 7 дней
- **Аудит-лог** — запись всех действий администратора с IP-адресом
- **GDPR-удаление** — полное стирание данных ученика (профиль + посещаемость)
- **Архивирование** — скрытие ученика без удаления данных
- **Ручная отметка** — «Болен» / «Уважительная причина»
- **Рассылка группе** — одно сообщение всем родителям через WA + TG + Email
- **Тестовые сообщения** — проверка WA, TG и Email из панели
- **Согласие на обработку ПД** — поле `consentDate`, визуальный индикатор, бланк печати
- **Автоочистка данных** — ночная задача по сроку хранения (`DATA_RETENTION_YEARS`)
- **Импорт из CSV** — загрузка списка учеников файлом
- **Поиск учеников** — фильтрация по имени ученика и родителя
- **Сортировка журнала** — клик по заголовку столбца
- **Дублирование группы** — скопировать настройки группы
- **Helmet.js** — CSP, X-Frame-Options, XSS-защита
- **Rate limiting** — `/scan` (10/мин), авторизация (10 попыток/15 мин), запись (60/мин)
- **`/health`** — health-check эндпоинт
- **`robots.txt`** — закрытие `/scan/`, `/admin/`, `/api/` от индексации
- **`privacy.html`** — полная политика конфиденциальности
- **`help.html`** — страница помощи для родителей
- **`Dockerfile`** + **`docker-compose.yml`** — контейнеризация
- **`migrate.js`** — миграция данных с версии 1.0.0 (JSON → SQLite)
- **UUID v4** для ID всех сущностей (вместо timestamp)

### Изменено
- `config.js` — `dotenv`, multi-provider WA конфигурация, SMTP настройки
- `whatsapp.js` — мультипровайдерный фасад с динамическим выбором
- `telegram.js` — полный рефакторинг, Webhook + Long Polling + `/stop` команда
- `public/index.html` — 10 вкладок, дашборд, аудит, Email, ошибки, confirm-диалоги, a11y
- `server.js` — Rate limiting, authGuard, QR группы, кабинет родителя, все новые маршруты
- `package.json` — добавлены `helmet`, `nodemailer`, `@whiskeysockets/baileys`

### Удалено
- `whatsapp-web.js` — убрана из зависимостей (не использовалась)
- `qrcode-terminal` — убрана из зависимостей

- Панель педагога (5 вкладок)
