## [1.1.0] — 2026-03-11

### Добавлено
- **SQLite вместо JSON** — база данных переведена на `sql.js` (чистый JS, без нативной компиляции)
- **Группы/классы** — управление несколькими классами, фильтрация во всех вкладках
- **Уведомление об опоздании** — автоматический расчёт по расписанию группы
- **Экспорт в Excel** — `.xlsx` с фильтром по группе и дате, форматированием
- **Статистика ученика** — сводка за 6 месяцев (всего/вовремя/опозданий/%)
- **Telegram Webhook** — автоматическая регистрация при наличии HTTPS-домена (fallback: Long Polling)
- **Дашборд** — вкладка «Сегодня» с живой сводкой + онбординг-чеклист
- **Аудит-лог** — запись всех действий администратора с IP-адресом
- **GDPR-удаление** — полное стирание данных ученика (профиль + посещаемость)
- **Архивирование** — скрытие ученика без удаления данных
- **Ручная отметка** — «Болен» / «Уважительная причина»
- **Рассылка группе** — одно сообщение всем родителям через WA + TG
- **Тестовые сообщения** — проверка WA и TG из панели без сканирования
- **Согласие на обработку ПД** — поле `consentDate`, визуальный индикатор
- **Автоочистка данных** — ночная задача по сроку хранения (`DATA_RETENTION_YEARS`)
- **Импорт из CSV** — загрузка списка учеников файлом
- **Поиск учеников** — фильтрация по имени ученика и родителя
- **Сортировка журнала** — клик по заголовку столбца
- **Дублирование группы** — скопировать настройки группы
- **Rate limiting** — `/scan/:id` ограничен 10 запросами/мин с одного IP
- **`/health`** — health-check эндпоинт
- **`robots.txt`** — закрытие `/scan/`, `/admin/`, `/api/` от индексации
- **`privacy.html`** — полная политика конфиденциальности
- **`help.html`** — страница помощи для родителей
- **`Dockerfile`** + **`docker-compose.yml`** — контейнеризация
- **`migrate.js`** — миграция данных с версии 1.0.0 (JSON → SQLite)
- **`.env.example`** — шаблон конфигурации с комментариями
- **Название школы** в тексте всех уведомлений (`SCHOOL_NAME`)
- **UUID v4** для ID всех сущностей (вместо timestamp)

### Изменено
- `config.js` теперь читает значения через `dotenv` из `.env`
- `whatsapp.js` — добавлен `sendDirect()` для тестовых сообщений
- `telegram.js` — полный рефакторинг, поддержка Webhook + handleWebhookUpdate()
- `public/index.html` — 9 вкладок вместо 7, дашборд, аудит, улучшенный UX
- `server.js` — Rate limiting, authGuard с IP, новые маршруты, cleanup IIFE
- `package.json` — добавлены `dotenv`, `express-rate-limit`, `uuid`; скрипты `start` и `dev`

### Удалено
- `whatsapp-web.js` — убрана из зависимостей (не использовалась)
- `qrcode-terminal` — убрана из зависимостей

### Исправлено
- Корректное отображение посещаемости при отсутствии группы

## [1.0.0] — 2025-06-01

### Добавлено
- Базовая система учёта посещаемости через QR-код
- WhatsApp уведомления через Green API с очередью и повторными попытками
- Telegram-бот (Long Polling, команды /start и /id)
- Параллельная отправка в WhatsApp и Telegram
- Защита от двойного скана (10 минут)
- Генерация и печать QR-кодов
- Защита `/admin` паролем (Basic Auth)
- JSON-файловая база данных
- Панель учителя (5 вкладок: Ученики, Посещаемость, Печать QR, WhatsApp, Telegram)
